# Планировщик участка

Учебный проект Осетровой Анны, ученицы 7 класса.

Интерактивное web-приложение для проектирования частного участка и дома. В нём можно настроить размеры участка, разместить дом, постройки и растения, проверить базовые отступы и посмотреть результат в 3D.

## Возможности

- 2D-планирование участка на сетке
- Размещение дома, построек и растений
- Проверка базовых норм и отступов
- Сохранение проекта в браузере
- 3D-просмотр результата
- Адаптация под компьютер и мобильные устройства

## Технологии

- React
- TypeScript
- Vite
- CSS
- three.js
